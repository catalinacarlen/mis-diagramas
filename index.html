Figuras estructuradas (Clase UML / Entidad ERD)

La primera línea es el título. Las siguientes son filas/miembros. Escribí una línea con -- para separar compartimentos (ej: atributos / métodos). Usá Shift+Enter para saltar de línea y Enter para confirmar.

Tu trabajo se autoguarda en este navegador. Para conservarlo o compartirlo, usá Guardar (.json) o exportá PNG/SVG.
